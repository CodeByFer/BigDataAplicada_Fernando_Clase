# BigDataAplicada_Fernando_Clase

GUIA DE INSTALACIÓN

PRIMERO EJECUTE ESTE COMANDO EN LA RAIZ DEL PROYECTO

``` sh
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

Notará que le sale (venv) al inicio de la linea de la terminal, lo que le indica que está en el enviroment del proyecto. Para salir utilice esto

```sh
deactivate
```

Una vez ya dispone de todo instalado y que haya salido de la terminal del entorno, puede usar esto para volver a entrar y así entrar en la Terminal del enviroment y ejecutar los archivos .py

``` sh
source venv/bin/activate
```
